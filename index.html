Product Strategy

I help software businesses focus on a vision and achieve goals by planning a roadmap. I measure the success of projects to make sure the most value is being delivered.

Wireframing and Prototyping

I create wireframes used to prototype possible design solutions. to prove any assumptions via user testing.

Front-End Software Development

Mobile App Design

I design mobile experiences and interfaces for native Android and iOS, and React Native apps.
